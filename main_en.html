HTTP Caching (from RFC 2616)

  • HTTP Cache: Stores copies of web pages to make loading faster and reduce bandwidth usage.
  • Cache-Control: Defines caching rules such as max-age and no-cache.
  • Expires: Shows the exact time when the cached copy becomes invalid.
  • Age: Indicates how long the cached copy has been stored.
  • ETag: Unique ID for a file version to check if content changed without downloading again.
